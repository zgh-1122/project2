## ✨ Features
- **Dynamic 15-Node Visa Directory Grid** with state-aware modal registration forms.
- **Intersection Observer Architecture** for smooth scroll-reveal loading animations.
- **Comprehensive Validation Checks** for legal names, alpha-numeric passports, and future travel indices.
- Dual-column structural dossier file uploads and compliance checklists.

## 🚀 Tech Stack
- **Frontend Framework:** React.js (v18+)
- **Styling:** Custom Vanilla CSS3 with premium grid/flexbox controls.
- **Deployment Hub:** Vercel

## 🛠️ Local Installation Setup
1. Clone the repository down to your local directory.
2. Execute `npm install` to synchronize core dependencies.
3. Run `npm start` to instantiate the local development framework at `localhost:3000`.
